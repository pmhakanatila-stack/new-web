# PEYZAJDER New Web

PEYZAJDER yeni web sitesi demo/yayın paketi.

## Notlar
- Tüm sayfa ve panel dosyaları UTF-8 karakter setiyle hazırlanmıştır.
- Görseller WebP akışına uyumlu olacak şekilde kurgulanmıştır.
- `api/public/*` dosyaları statik demo API çıktılarıdır.
- Yerel geliştirme için proje kökündeki `server.mjs` outputs klasörünü çalıştırır.
